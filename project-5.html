Color scheme generator

A program that generates color scheme depending on the users selection.

Live Link

Project Overview

As a developer, designer, or anyone working with colors, we sometimes struggle to find the perfect color codes or schemes for our projects. While it may be straightforward for some, it can be time-consuming for others.

The color scheme generator simplifies this process by allowing users to select a starting color and choose a mode. The app then generates five colors that match the user's criteria.

Users can easily copy the desired color code with just a click.

This app is built using HTML, CSS, JavaScript, and a color API, and it's fully responsive for all devices.
